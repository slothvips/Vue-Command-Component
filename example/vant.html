Vant示例 ​

本页面展示如何使用Vant UI组件库的适配器。

要使用Vant的命令式组件，需要安装@vue-cmd/core和@vue-cmd/vant两个包：

bash
# 使用npm
npm install @vue-cmd/core @vue-cmd/vant vant

# 使用pnpm
pnpm add @vue-cmd/core @vue-cmd/vant vant

# 使用yarn
yarn add @vue-cmd/core @vue-cmd/vant vant
